# QtMarkdown

一个简单的基于Qt的Markdown解析器和编辑器。

## Feature

- [x] Markdown的解析和渲染
- [x] CheckBox交互，点击选中或者选中
- [x] Link交互，如用浏览器打开链接
- [x] Image交互，如播放GIF动画
- [ ] Image地址编辑
- [ ] 粘贴图片
- [ ] Redo & Undo
